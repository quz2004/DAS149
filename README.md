# DAS149 — Career Development in Data Science

**Fei Tian College · Spring 2026 · Dr. Zheng Qu**

---

## Course Description

A seminar-style course (5–6 sessions) that equips students with practical knowledge and tools for launching a data science career. Topics span industry trends, job roles, portfolio building, and strategic career planning. The course culminates in a student portfolio presentation.

## Learning Objectives

- Understand the data science landscape and its diverse career paths
- Build and present a professional online profile and project portfolio
- Identify the skills and competencies required for specific data science roles
- Develop a personal, actionable career action plan

## Key Topics

- **GitHub & SSH setup** — version control for data scientists
- **Data storytelling** — writing READMEs that communicate impact
- **Personal branding** — logos, profile pages, pinned projects
- **AI-assisted workflows** — using Antigravity / Gemini Pro effectively
- **Career planning** — roles, skills maps, and portfolio strategy

## Assessment

This is a **Pass / No Pass** course. Students must:
- Attend all sessions
- Submit all assignments on time
- Give a 5-minute final portfolio presentation
